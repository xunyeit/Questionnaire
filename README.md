讯业软件开发/讯业问卷-java问卷调差系统

## 一、产品概述
**“****问卷****”是一款聚焦于****个人健康评估与生活方式干预****的数字化健康管理应用。它以“****测一测，懂自己****”为核心理念，通过科学设计的在线问卷、互动测评和健康科普内容，帮助用户全面了解自身健康状况、识别潜在风险因素，并提供个性化建议与线下活动支持。**

**该应用不仅是一个信息收集工具，更是一个融合了自我认知、行为引导、社群互动与服务转化于一体的综合性健康管理平台。用户可通过参与测评获取专属健康报告，阅读权威文章提升健康素养，并报名参加身心调养类活动，实现从“被动知晓”到“主动改善”的闭环管理。**

## 二、核心功能
### 1. **健康测评问卷（动态评估入口）**
+ **用户进入首页后可点击“马上测吧”，启动一系列标准化健康自评问卷。**
+ **问卷内容涵盖多个维度：**
    - **基本信息****：年龄、性别、职业等**
    - **睡眠质量****：每晚平均睡眠时长（如6.5–7.5小时）**
    - **吸烟饮酒习惯****：是否吸烟、饮酒频率及量级**
    - **心理压力状态****：近期情绪波动、焦虑程度**
    - **重大生活变故****：是否经历亲人离世、失业、离婚、财务危机等**
    - **宠物饲养情况****：是否有宠物（影响生活方式与情绪）**
    - **社交频率：与亲友聚会频次（反映社会支持系统）**

### 2. **健康科普文章（知识赋能模块）**
+ **在完成问卷或浏览主页时，用户可查看“文章”板块中的专业健康内容。**
+ **示例文章标题：“日常生活中，加速衰老进程的前8个普遍因素”**
+ **内容结构清晰，包含：**
    - **图文并茂的可视化图表（如Lifestyle环形图）**
    - **科学依据说明：分析饮食不当、缺乏运动、长期久坐对身体的影响**
    - **风险提示：如高盐高糖饮食导致高血压、糖尿病风险上升**
    - **建议措施：提倡均衡膳食、规律作息、适度锻炼**
+ **支持收藏、分享至微信朋友圈或群聊，便于传播健康理念。**

### 3. **活动报名系统（行为激励与服务延伸）**
#### （1）活动类型丰富，覆盖多场景需求：
+ **清体排毒 / 正念冥想 / 颂钵疗愈****：  
****提供养生类体验活动，帮助用户缓解压力、调节情绪、促进身体代谢。**
+ **逆龄饮食指导****：  
****传递科学抗衰饮食理念，教授如何搭配营养餐食，从饮食层面延缓衰老。**
+ **聚会社交活动****：  
****组织线下社交类主题活动，增强用户之间的互动交流，在轻松氛围中建立健康生活圈层。**

#### （2）报名流程便捷高效：
+ **用户点击“报名”按钮后，进入订单填写页。**
+ **需填写：**
    - **姓名**
    - **联系电话**
    - **可选优惠券使用**
+ **支付方式支持微信支付，实现快速完成注册。**
+ **系统自动保存订单信息，方便后续通知与跟进。**

### 4. **个人中心与积分体系（用户成长闭环）**
+ **用户可在“个人中心”查看以下内容：**
    - **我的分享****：已分享的文章或活动链接记录**
    - **我的积分****：参与测评、阅读文章、转发分享等行为累积的积分**
    - **我的钱包****：可用于支付活动费用或兑换礼品的虚拟账户**
    - **我的测试****：历史测评记录，支持回顾与对比**
    - **清理缓存****：优化App运行速度**
    - **开始测试 / 分享活动：快捷入口，提升活跃度**

## 三、产品价值
### **用户价值：实现自我认知与健康管理升级**
+ **精准洞察健康状态****：通过科学问卷发现潜在健康隐患，早干预、早预防。**
+ **获得个性化建议****：根据测评结果推荐适合的饮食方案、运动计划或心理调节方法。**
+ **享受优质线下服务****：参与高质量养生活动，满足精神与身体双重需求。**
+ **构建健康社交圈****：通过活动结识志同道合的朋友，形成正向影响群体。**
+ **低成本获取专业健康知识：无需去医院挂号，即可阅读权威科普文章，提升健康素养。**

## 架构特点
### 架构特性
+ **前后端分离架构，独立开发，符合主流开发模式**
+ **前端以Vue3+Vite为主技术，AntdV为UI界面框架**
+ **后端SpringBoot3为基础，MybatisPlus为数据操作框架，Redis为缓存框架**
+ **Maven多模块管理，插件化开发，方便安装、卸载、升级，降低耦合**
+ **业务模块与API抽离，模块之间便捷引用**
+ **数据库设计精巧，字段规范、易于扩展**
+ **支持国产密码算法加解密，等保测评国产项目无压力**
+ **支持MYSQL、ORACLE、SQLSERVER、PGSQL等主流标准结构式数据库**
+ **支持达梦、人大金仓、南大通用、九有、瀚高、虚谷数据库等国产数据库**
+ **支持中创、宝蓝德、东方通等中间件**
+ **支持Windows、Linux操作系统、国产操作系统部署**

### 功能特性
+ **完善的系统基础功能，满足使用需求，避免重复造轮子**
+ **支持本地文件、阿里云文件、腾讯云文件、MINIO文件上传**
+ **支持本地邮件、阿里云邮件、腾讯云邮件发送**
+ **支持阿里云短信、腾讯云短信发送**
+ **B、C端双账号认证体系，会话治理各自独立**
+ **完善的登录日志、操作日志、异常日志等审计功能**
+ **完善的会话监控、数据源监控、系统监控等必备监控功能**
+ **支持组织机构、权限管理、定时任务、系统配置等基础功能**

### 安全特性
+ **采用SaToken轻量级 Java 权限认证框架，功能强大、学习成本低**
+ **支持登录认证、权限认证、单点登录、三方登录、OAuth2.0等认证模式**
+ **增强的RBAC权限设计，资源于接口独立授权，更加灵活**
+ **支持按钮级别细粒度独立授权，界面按钮动态展示**
+ **支持API接口注解式、路由拦截式鉴权，防止越界访问**
+ **独创的数据范围机制，每个接口都可以配置不同数据范围**
+ **支持限流防抖，防重复提交，有效阻止脏数据产生**
+ **密码、手机、身份证号等使用国密算法加密传输、加密存储，数据更安全**
+ **操作日志使用SM2进行完整性保护，满足安全审计要求**

### 界面特性
+ **Vue3 + Vite为基础，AntdV为界面UI框架，视觉风格清新简洁**
+ **精细化设计，注重界面的每一处细节，操作轻松友好**
+ **暗黑风格、经典菜单、双排菜单、多页签、目录坞、主题切换等功能应有尽有**
+ **统一的网络框架、API接口拦截框架，拿来即可上手**

+ **2 核 2 G 100MB 3M带宽 为最低配置，建议 2 核 4G 100Mb 5M带宽。**
+ **Node.js 14x**
+ **Java1.8**
+ **Mysql 5.7**
+ **Redis 6.2.8**
+ **阿里云 OSS – 可更换其它。**
